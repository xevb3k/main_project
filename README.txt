Тест добавления товаров в корзину в интернет-магазине М-Видео.

1. Переход в каталог смартфонов
2. Активация фильтра по бренду (нескольким брендам), список брендов для фильтра можно передать списком
3. Если карточки товаров выставлены сеткой - переключение на отображение списком
4. Создание списка карточек товара в формате (название, цена), исключая недоступные для добавления в корзину
5. Добавление в корзину случайного количества товаров в случайном порядке
6. Переход в корзину
7. Создание списка товров в корзине
8. Тест пройден успешно если множество добавленных в корзину товаров является подмножеством товаров в корзине

Скрипт закрывает следующие всплывающие окна для стабильной работы:
1. сообщение о геолокации
2. сообщение о куках
3. всплывающее окно корзины
